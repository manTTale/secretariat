Tutorial on Create Document

The Create Document feature could be accessed by clicking on the "Create Document" button on the top right corner.

  1. This functionality allows secretaries use drag and drop technology for creating custom documents by clicking the "Create new form template" button
  2. For each document the secretary wants to create they need to provide a
    • Code - used by the application for the identification of the document
    • Name - which represents the name of the document created
    • Description - which could indicate some information about the document
    • Define form - here the secretary can use the features available on the left to create, using drag and drop, the document.
      • After dragging a dropping each element the secretary will be prompted to give a title for the field, and to complete some other information to better suit their needs.
  3. After the document it's created like the secretary intended they need to press the "Save Form Template" button in order to save in the application the form.
  4. The user can then go back to the Create Document page where it can view informations about the created documents while also having the capability of editing and deleting the documents
