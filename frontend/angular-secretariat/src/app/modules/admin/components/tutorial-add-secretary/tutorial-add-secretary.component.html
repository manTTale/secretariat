Tutorial on Add a new secretary

The Add a new secretary feature could be accessed by clicking on the "Add a new secretary" button on the top right corner.

  1. This functionality allows administrators to create a new secretary account by introducing their data
  2. The administrator will be prompted to introduce the secretary's last name as well as their first name
  3. The button "Submit" will be then pressed to create the secretary account
  4. The newly created secretary can be viewed by pressing the "Manage secretaries" button on the top right corner
