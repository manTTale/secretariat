Tutorial on Add a new student

The Add a new student feature could be accessed by clicking on the "Add a new student" button on the top right corner.

  1. This functionality allows administrators to create a new student account by introducing their data
  2. The administrator will be prompted to introduce the student's last name as well as their first name, specialization, group number and matriculation number
  3. The button "Submit" will be then pressed to create the student account
  4. The newly created student can be viewed by pressing the "Manage students" button on the top right corner
