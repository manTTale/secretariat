Tutorial on Manage Students

The Manage Students feature could be accessed by clicking on the "Manage Students" button on the top right corner.

  1. This functionality allows administrators to see the whole list of students created within the app.
  2. It will list the id, first name, last name, specialization, group number, Student Id for every student registered.
  3. On the column "Action" on each student row there are present two buttons.
    • The trashcan button can be used to delete a student from the system.
    • The other icon is used for when there's a need to edit the information of some student, for example in the case of a mistake.
