Tutorial on Manage Secretaries

The Manage Secretaries feature could be accessed by clicking on the "Manage Secretaries" button on the top right corner.

  1. This functionality allows administrators to see the whole list of secretaries created within the app.
  2. It will list the id, first name, last name, specialization, group number, Student Id for every secretart registered.
  3. On the column "Action" on each secretary row there are present two buttons.
    • The trashcan button can be used to delete a secretary from the system.
    • The other icon is used for when there's a need to edit the information of some secretary, for example in the case of a mistake.
